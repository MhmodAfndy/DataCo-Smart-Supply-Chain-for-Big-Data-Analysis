# DataCo Smart Supply Chain for Big Data Analysis 📊

## 📋 نبذة عن المشروع

مشروع تحليل شامل لسلسلة التوريد الذكية لشركة DataCo باستخدام تقنيات البيانات الكبيرة (Big Data). يتضمن المشروع تحليلات متقدمة، لوحات معلومات تفاعلية، والعديد من التصورات البيانية.

## 📁 هيكل المشروع

```

├── 📁 analysis/
│   └── Python Analysis For Supply Chain.ipynb
│       └── تحليلات شاملة باستخدام Python
│
├── 📁 dashboards/
│   └── dashboard.pbix
│       └── لوحة معلومات تفاعلية في Power BI
│
├── 📁 visualizations/
│   ├── Market & Revenue.jpg

│   └── Shipping & Orders.jpg

│

│   └── Presentation Storytelling.pptx
│       └── عرض شامل لقصة البيانات

```

## 📂 محتويات المجلدات

### 📊 Analysis (`analysis/`)
- **Python Analysis For Supply Chain.ipynb**: دفتر Jupyter يحتوي على:
  - استكشاف البيانات (EDA)
  - تحليلات متقدمة
  - معالجة البيانات
  - الرؤى الأساسية

### 📈 Dashboards (`dashboards/`)
- **dashboard.pbix**: لوحة معلومات Power BI تفاعلية تتضمن:
  - مؤشرات أداء رئيسية (KPIs)
  - رسوم بيانية ديناميكية
  - تقارير فترات زمنية

### 🎨 Visualizations (`visualizations/`)
- **Market & Revenue.jpg**: تصور السوق والإيرادات
- **Sales & Profit.jpg**: تصور المبيعات والأرباح
- **Shipping & Orders.jpg**: تصور الشحنات والطلبات

### 📑 Presentations (`presentations/`)
- **Presentation Storytelling.pptx**: عرض شامل يروي قصة البيانات بشكل احترافي

### 💾 Data (`data/`)
مخصص للبيانات:
- `raw/` - البيانات الخام الأصلية
- `processed/` - البيانات المعالجة والنظيفة

## 🛠️ الأدوات المستخدمة

| الأداة | الوصف |
|-------|-------|
| 🐍 **Python** | تحليل البيانات وعمليات المعالجة |
| 📔 **Jupyter Notebook** | بيئة التطوير التفاعلية |

## 🚀 كيفية البدء

### المتطلبات:
- Python 3.7+
- Jupyter Notebook
- Power BI Desktop (اختياري)
- المكتبات المطلوبة (راجع `requirements.txt`)

### التثبيت:
```bash
# استنساخ المستودع
git clone https://github.com/MhmodAfndy/DataCo-Smart-Supply-Chain-for-Big-Data-Analysis.git

# الدخول للمستودع
cd DataCo-Smart-Supply-Chain-for-Big-Data-Analysis

# تثبيت المكتبات
pip install -r requirements.txt
```

### تشغيل التحليلات:
```bash
jupyter notebook analysis/Python\ Analysis\ For\ Supply\ Chain.ipynb
```
